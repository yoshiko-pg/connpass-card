# 参加者の名は。

Connpass や meetup の URL から参加者の名札を印刷できます。

スマートフォンは横持ちでどうぞ。

https://yoshiko-pg.github.io/your-name/

Created by @yoshiko-pg https://yoshiko-pg.github.io/

## Requirements

* Node.js `7.2.1`

（バージョン 8 だとデプロイに失敗する）

## Setup

```
yarn install
```

## Development

```
yarn run serve:api
```

```
yarn run serve:ng
```

open http:localhost:4200
